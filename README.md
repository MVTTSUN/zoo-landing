## Описание проекта

Сайт имеет слайдеры карточек животных и отзывов, также реализован переход на вкладку Donate с интерактивным рассчетом стоимости доната.

## Технологии, использованные в проекте

* HTML
* CSS
* SASS
* JavaScript
